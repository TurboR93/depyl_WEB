# Depyl - Accesso all'acqua per tutti

Benvenuto nel repository del sito web di **Depyl**, azienda che opera con l'obiettivo di garantire l'accesso all'acqua pulita in tutto il mondo.

## 💧 La Nostra Missione
Vendere impianti per la depurazione dell'acqua e destinare parte dei ricavi a progetti che portino l'acqua potabile nelle aree del mondo dove è ancora scarsa o difficilmente accessibile.

## 🌍 La Nostra Visione
Un mondo che sta iniziando a fare viaggi turistici nello spazio non può essere lo stesso mondo in cui l'acqua è ancora un bene inaccessibile per molte persone. Vogliamo creare un mondo in cui **tutti abbiano accesso alla risorsa più preziosa: l'acqua.**

## ⚙️ Come Funziona
Offriamo soluzioni diversificate e a basso impatto ambientale:
- **Soluzioni B2C**: Impianti di depurazione per abitazioni private con filtri scomponibili per facilitare il riciclo.
- **Soluzioni B2B**: Sistemi di depurazione per ristoranti e attività commerciali con tecnologie avanzate.
- **Sostenibilità**: Filtri domestici progettati per essere scomponibili con una corretta raccolta differenziata.
- **Impatto Sociale**: Una parte dei nostri profitti finanzia progetti di accesso all'acqua in Paesi in via di sviluppo.

## 🚀 Sviluppo Locale
Questo progetto è un sito web statico (HTML, CSS, JS). Per visualizarlo in locale:

1. Clona il repository o naviga nella cartella del progetto.
2. Avvia un server locale con Python:
   ```bash
   python3 -m http.server
   ```
3. Apri il browser all'indirizzo `http://localhost:8000` per visualizzare il sito.
