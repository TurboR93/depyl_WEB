# Depyl - Accesso all'acqua per tutti 💧

Sito web di **Depyl**, azienda impegnata nella depurazione dell'acqua e nella lotta alla crisi idrica globale.

## 🌍 La Nostra Missione
Vendere impianti per la depurazione dell'acqua e destinare parte dei ricavi a progetti che portino l'acqua potabile nelle aree del mondo dove è ancora scarsa o difficilmente accessibile.

## 🌊 La Nostra Visione
Creare un mondo in cui **tutti abbiano accesso alla risorsa più preziosa: l'acqua.**

## ⚙️ Soluzioni
- **B2C** — Impianti di depurazione per abitazioni private con filtri scomponibili per facilitare il riciclo.
- **B2B** — Sistemi di depurazione per ristoranti e attività commerciali con tecnologie avanzate.
- **Sostenibilità** — Filtri progettati per essere scomponibili e riciclabili al 100%.
- **Impatto Sociale** — Una parte dei profitti finanzia progetti di accesso all'acqua in Paesi in via di sviluppo.

## 🛠️ Tech Stack
- **HTML5** — Struttura semantica con SEO ottimizzato (meta tags, Open Graph, heading hierarchy)
- **CSS3** — Design premium con variabili CSS, glassmorphism, animazioni scroll-triggered, responsive design
- **JavaScript** — Smooth scrolling, IntersectionObserver animations, animated counters, mobile menu
- **Google Fonts** — Inter (sans-serif) + Playfair Display (serif)
- **Unsplash** — Immagini placeholder gratuite (da sostituire con foto proprietarie)

## 🎨 Design
UI ispirata a design premium moderni con:
- Tema scuro con accenti sky-blue (`#0ea5e9`)
- Header con effetto glassmorphism (backdrop-filter blur)
- Tipografia bold con font serif per i titoli
- Cards con bordi luminosi e hover glow
- Animazioni scroll-reveal con stagger
- Layout responsive (desktop, tablet, mobile con hamburger menu)

## 🚀 Sviluppo Locale
